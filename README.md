# Sustainable Living App

A lightweight Expo + React Native starter for exploring simple sustainable living ideas.

## Prerequisites
- Node.js 20+
- npm 9+

## Setup
```bash
npm install
```

## Development
```bash
npm start
```

This launches the Expo development server so you can load the app on a simulator or Expo Go.

## Quality
```bash
npm run lint
npm run typecheck
npm test
```

## Scripts
| Command | Description |
| --- | --- |
| `npm start` | Start Expo dev server |
| `npm run android` | Build & run Android project |
| `npm run ios` | Build & run iOS project |
| `npm run web` | Run the Expo web preview |
| `npm run lint` | ESLint in strict mode |
| `npm run typecheck` | TypeScript `--noEmit` check |
| `npm test` | Jest unit tests |
| `npm run format` | Prettier check |
| `npm run format:write` | Prettier write |

## Notes
- Icons and splash screens rely on Expo defaults to keep the repository binary-free.
- Add a `package-lock.json` after the first successful `npm install` in your environment.
